# 📍 Google Maps Business Analysis Dashboard

A data science project that analyzes local businesses in Bhagalpur using Google Maps business data and identifies opportunities for digital transformation.

---

## 🚀 Project Objective

The main objective of this project is to:

- Analyze local businesses in Bhagalpur.
- Find businesses that do not have websites.
- Identify high-opportunity businesses for digital services.
- Visualize business locations and categories using an interactive dashboard.

---

## 📊 Features

✅ Interactive Dashboard using Streamlit

✅ Business Category Analysis

✅ Website vs No Website Analysis

✅ Top Businesses by Rating and Reviews

✅ High Opportunity Business Identification

✅ Business Location Visualization using Folium Maps

✅ Search and Filter Functionality

✅ Download Filtered Data as CSV

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/your-username/Google_Maps_Business_Analysis.git
cd Google_Maps_Business_Analysis
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the dashboard:

```bash
streamlit run src/dashboard.py
```

## 📌 Key Insights

- More than 99% of businesses do not have websites.
- General Shops and Kirana Stores dominate the market.
- Most businesses have low digital presence.
- There is a huge opportunity for website development and digital marketing services.
